ol3-turf.js

A Javascript library that provides a Turf toolbar for OpenLayers.

Download »

Goal

The main goal of ol3-turf is to provide a easy-to-use library that combines the mapping capabilities of OpenLayers with the geospatial analysis tools of Turf.

Features

An out-of-the-box solution to display an OpenLayers toolbar with access to Turf commands and popup forms and messages to provide inputs and outputs.

Configurable

The user can control the styling of the toolbar and its controls by provding custom style sheets, as well a custom handler if the default handler is not sufficient.
