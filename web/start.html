Getting Started

Setup

The ol3-turf library has two dependencies that must be included, OpenLayers 3 and turf. Therefore, the following is the minimum needed to set up an ol3-turf web application. 

    <link href="path/to/ol3/ol.css" rel="stylesheet" type="text/css"/>
    <link href="path/to/turf/turf.min.css" rel="stylesheet" type="text/css" />
    <link href="path/to/ol3-turf/ol3-turf.min.css" rel="stylesheet" type="text/css" />

    <script src="path/to/ol3/ol.js" type="text/javascript"></script>
    <script src="path/to/turf/turf.min.js" type="text/javascript"></script>
    <script src="path/to/ol3-turf/ol3-turf.min.js" type="text/javascript"></script>

Usage

Using the ol3-turf library is as simple as creating a new instance of a ol.control.Turf control and then adding it to an existing ol.Map. The default toolbar provides reasonable defaults for it's layout and commands. By default the toolbar is displayed across the top with all available turf commands enabled. For commands requiring inputs a popup form is displayed. If the turf command returns a feature it is added to the map. If a value is returned it is displayed in a popup message.

Example

Below is the standard toolbar that is displayed when no options are provided. 

    var toolbar = new ol.control.Turf();
    var map = new ol.Map({...});
    map.addControl(toolbar);

Options

The ol.control.Turf(options) constructor takes an optional argument that allows the user to override the default settings like the styling of the toolbar and providing custom processing of turf command outputs. For a complete description see the ol.control.Turf documentation. 

    options = {ol3turf: {
        controls : ["area", ...],   // list of controls to add to toolbar
        handler  : {                // callback to handle turf commands output
            callback : function(name, inputs, output) {...}
        },
        prefix   : "ol3-turf-foo",  // prefix to apply to toolbar annd control IDs.
        style    : "ol3-turf-bar"   // style to apply to toolbar
    }};
