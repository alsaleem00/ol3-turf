Demos

Stand alone examples

The following links provide stand alone examples using a standard toolbar, multiple toolbars, and integrating the toolbar with the OpenLayers Explorer (olexp) library.

Standard toolbar

This example shows the standard toolbar with the default settings. For example, selecting a polygon and selecting the "Measure Area" control will display its area in squared meters. 

    <script type="text/javascript">
        var map = new ol.Map({...});
        var toolbar = new ol.control.Turf();
        map.addControl(toolbar);
    </script>

Multiple toolbars

This example shows how to use the prefix and style options to use multiple toolbars. 

    <style>
        .ol3-turf-toolbar-measurement
        {
            background-color: initial;
            left: 4.5em;
            padding: initial;
            top: 0.5em;
        }
        .ol3-turf-toolbar-grids
        {
            background-color: initial;
            left: 0.5em;
            padding: initial;
            top: 6.5em;
        }
        .ol3-turf-toolbar-grids .ol3-turf-control
        {
            float: none;
        }
    </style>
    ...
    <script type="text/javascript">
        var map = new ol.Map({...});
        var toolbar = new ol.control.Turf({ol3turf: {
            controls: ol3turf.toolbars.measurement(),
            style: "ol3-turf-toolbar-measurement"
        }});
        map.addControl(toolbar);

        var toolfoo = new ol.control.Turf({ol3turf: {
            controls: ol3turf.toolbars.grids(),
            prefix: "ol3-turf-grids",
            style: "ol3-turf-toolbar-grids"}});
        map.addControl(toolfoo);
    </script>

Custom handler

This example shows how to pass an optional constructor argument to provide a custom handler for proccesing turf command outputs. For example, selecting a polygon and selecting the "Measure Area" control will display a custom alert message. 

    <script type="text/javascript">
        var map = new ol.Map({...});
        var toolbar = new ol.control.Turf({
            ol3turf: {
                handler: {
                    callback: function(name, inputs, output) {
                        alert("Custom handler for command '" + name + "'");
                    }
                }
            }
        });
        map.addControl(toolbar);
    </script>
